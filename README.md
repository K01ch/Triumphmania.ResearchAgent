## 📌 Описание

`Triumphmania.ResearchAgent` — это автономный исполняемый модуль, разработанный для глубинного исследования архитектуры Windows, методов межпроцессного взаимодействия и механизмов персистентности. Инструмент предоставляет исследовательский стек для изучения поведения защитных решений в условиях активного противодействия.

**Ключевые направления:**
- Анализ реакций антивирусных движков на нестандартные паттерны поведения
- Исследование методов сокрытия активности в пользовательском режиме
- Тестирование механизмов обнаружения на основе системных вызовов
- Оценка устойчивости защитных систем к длительным нагрузкам

## 🧩 Функциональные возможности

### Системная интеграция
- **Самокопирование** — автоматическое размещение исполняемого файла в системную директорию
- **Персистентность** — создание ключа автозагрузки в реестре
- **Механизм восстановления** — фоновый поток для восстановления компонентов при удалении
- **Повышение привилегий** — запрос административных прав через UAC

### Сетевая активность
- **Сканирование локальных сегментов** — проверка доступности портов в диапазоне 10000-60000
- **Экфильтрация данных** — отправка системной информации на управляющий сервер
- **Шифрование трафика** — симметричное шифрование AES-256 для передаваемых данных

### Мониторинговые модули
- **Сбор системной информации** — имя хоста, пользователь, версия ОС, список активных процессов
- **Создание скриншотов** — периодический захват рабочего стола с кодированием в Base64

### Механизмы низкоуровневого доступа
- **Инжекция кода** — внедрение библиотеки в адресное пространство explorer.exe
- **Работа с памятью** — использование VirtualAllocEx, WriteProcessMemory, CreateRemoteThread
- **Анти-отладка** — детектирование отладчиков через IsDebuggerPresent

## 🛠️ Сборка и запуск

### Требования
- Visual Studio 2017+ или .NET Framework SDK
- Компилятор csc.exe (входит в состав SDK)

### Инструкция по сборке

```bash
# Компиляция в консольное приложение
csc /target:exe /out:Triumphmania.ResearchAgent.exe Program.cs

# Компиляция с отключением консольного окна (рекомендуется)
csc /target:winexe /out:Triumphmania.ResearchAgent.exe Program.cs
